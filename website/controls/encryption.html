Cryptographic controls protect data confidentiality and integrity both at rest and in transit. All algorithms are drawn from the approved list in the Cryptography Policy. FIPS 140-2 validated modules are used in CUI processing environments per CMMC L2 requirements.

Control ID Control Name Implementation SOC 2 ISO 27001 CMMC L1 CMMC L2 ISO/SAE 21434 Status

SC-12 Cryptographic Key Establishment & Management Keys managed in cloud KMS (AWS KMS / GCP KMS); HSM-backed for CUI environments; data encryption keys rotated annually; TLS certificates rotated ≤90 days. CC6.1 5.31 — SC.3.187 8.9 🟡 Partial
